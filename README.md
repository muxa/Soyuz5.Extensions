Soyuz5.Extensions
=================

A set of .NET extension methods for every day use.

Includes extensions for:

* DateTime
 * RoundToSecond()
 * RoundToMinute()
 * GetFirstDayOfWeek(...)
 * GetFirstDayOfMonth()
 * GetLastDayOfMonth()
 * GetMonthsSpanned()
 * IsSameWeek()
 * IsSameMonth()
 * IsSameYear()
 * IsWeekend()
 * GetWeekNumber()
 * GetFirstMondayOfYear()
* String
 * Limit()
 * TitleCaseToWords()
 * ReplaceTokens()
 * WrapLines()
 * JoinReadable()
* Enum
 * GetFlags()
 * GetEnumDescription()
 * GetEnumBindableList()
* And more
